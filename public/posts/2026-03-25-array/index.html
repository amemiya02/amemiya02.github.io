53. 最大子数组和 - Mid

核心思路：动态规划（Kadane 算法）

状态定义：

👉 dp[i] = 以 nums[i] 结尾的最大子数组和

转移方程：

dp[i]=max⁡(nums[i],dp[i−1]+nums[i]) dp[i] = \max(nums[i], dp[i-1] + nums[i]) dp[i]=max(nums[i],dp[i−1]+nums[i])

代码实现（Python） 

def maxSubArray(nums):
    pre = 0
    ans = nums[0]

    for num in nums:
        pre = max(pre + num, num)
        ans = max(ans, pre)

    return ans

复杂度分析

  • 时间复杂度: O(n)O(n)O(n)
  • 空间复杂度: O(1)O(1)O(1)

56. 合并区间 - Mid

核心思路：排序 + 贪心

👉 先排序，再线性合并

代码实现（Python） 

def merge(intervals):
    intervals.sort(key=lambda x: x[0])
    res = []

    for start, end in intervals:
        if not res or start > res[-1][1]:
            res.append([start, end])
        else:
            res[-1][1] = max(res[-1][1], end)

    return res

复杂度分析

  • 时间复杂度: O(nlog⁡n)O(n \log n)O(nlogn)
  • 空间复杂度: O(n)O(n)O(n)

189. 轮转数组 - Mid

核心思路：三次翻转

👉 整体翻转 + 局部翻转

代码实现（Python） 

def rotate(nums, k):
    n = len(nums)
    k %= n

    def reverse(l, r):
        while l < r:
            nums[l], nums[r] = nums[r], nums[l]
            l += 1
            r -= 1

    reverse(0, n - 1)
    reverse(0, k - 1)
    reverse(k, n - 1)

复杂度分析

  • 时间复杂度: O(n)O(n)O(n)
  • 空间复杂度: O(1)O(1)O(1)

238. 除了自身以外数组的乘积 - Mid

核心思路：前缀积 + 后缀积

👉 答案 = 左边积 × 右边积

代码实现（Python） 

def productExceptSelf(nums):
    n = len(nums)
    res = [1] * n

    # 后缀积
    suffix = 1
    for i in range(n - 1, -1, -1):
        res[i] = suffix
        suffix *= nums[i]

    # 前缀积
    prefix = 1
    for i in range(n):
        res[i] *= prefix
        prefix *= nums[i]

    return res

复杂度分析

  • 时间复杂度: O(n)O(n)O(n)
  • 空间复杂度: O(1)O(1)O(1)（不算输出数组）

41. 缺失的第一个正数 - Hard

核心思路：原地哈希（索引映射）

👉 数值 x 应该放在 index = x - 1

代码实现（Python） 

def firstMissingPositive(nums):
    n = len(nums)

    # 1. 原地放置
    for i in range(n):
        while 1 <= nums[i] <= n and nums[nums[i] - 1] != nums[i]:
            idx = nums[i] - 1
            nums[i], nums[idx] = nums[idx], nums[i]

    # 2. 查找缺失
    for i in range(n):
        if nums[i] != i + 1:
            return i + 1

    return n + 1

复杂度分析

  • 时间复杂度: O(n)O(n)O(n)
  • 空间复杂度: O(1)O(1)O(1)
