多维动态规划 (Multi-dimensional Dynamic Programming)

62. 不同路径 - Medium

核心思路

机器人只能向下或向右，这意味着到达 (i,j)(i, j)(i,j) 的唯一途径是来自 (i−1,j)(i-1, j)(i−1,j) 或 (i,j−1)(i, j-1)(i,j−1)。

  1. 递推公式：dp[i][j]=dp[i−1][j]+dp[i][j−1]dp[i][j] = dp[i-1][j] + dp[i][j-1]dp[i][j]=dp[i−1][j]+dp[i][j−1]。
  2. 空间优化：由于当前行只依赖于上一行和左侧元素，我们可以将二维数组压缩为一维。

代码实现 

class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        # 空间优化：只用一维数组
        # 初始状态：第一行全为 1（因为只能一直往右走）
        dp = [1] * n

        for i in range(1, m):
            for j in range(1, n):
                # dp[j] 还是上一行的值，dp[j-1] 是当前行左侧的值
                dp[j] += dp[j-1]

        return dp[-1]

复杂度分析

  • 时间复杂度：O(m×n)O(m \times n)O(m×n)。
  • 空间复杂度：O(n)O(n)O(n)。

64. 最小路径和 - Medium

核心思路

逻辑与“不同路径”几乎一致，只是从“加法”变成了“取最小值”。

  • 状态方程：dp[i][j]=min⁡(dp[i−1][j],dp[i][j−1])+grid[i][j]dp[i][j] = \min(dp[i-1][j], dp[i][j-1]) + \text{grid}[i][j]dp[i][j]=min(dp[i−1][j],dp[i][j−1])+grid[i][j]。
  • 技巧：我们可以直接在原数组 grid 上更新，从而实现 O(1)O(1)O(1) 的额外空间。

代码实现 

class Solution:
    def minPathSum(self, grid: list[list[int]]) -> int:
        m, n = len(grid), len(grid[0])

        for i in range(m):
            for j in range(n):
                if i == 0 and j == 0: continue
                elif i == 0: # 第一行，只能从左边来
                    grid[i][j] += grid[i][j-1]
                elif j == 0: # 第一列，只能从上面来
                    grid[i][j] += grid[i-1][j]
                else:
                    grid[i][j] += min(grid[i-1][j], grid[i][j-1])

        return grid[-1][-1]

复杂度分析

  • 时间复杂度：O(m×n)O(m \times n)O(m×n)。
  • 空间复杂度：O(1)O(1)O(1)（原地修改）。

5. 最长回文子串 - Medium

核心思路

判断一个子串是否回文：如果 s[i]==s[j]s[i] == s[j]s[i]==s[j]，且中间部分 s[i+1:j−1]s[i+1 : j-1]s[i+1:j−1] 也是回文，那么 s[i:j+1]s[i:j+1]s[i:j+1] 就是回文。

  • 状态定义：dp[i][j]dp[i][j]dp[i][j] 表示 s[i..j]s[i..j]s[i..j] 是否为回文。
  • 遍历技巧：子串长度从 2 开始递增，确保计算长子串时，短子串的结果已确定。

代码实现 

class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        if n < 2: return s

        dp = [[False] * n for _ in range(n)]
        for i in range(n): dp[i][i] = True

        max_len, start = 1, 0

        # L 为子串长度
        for L in range(2, n + 1):
            for i in range(n):
                j = L + i - 1
                if j >= n: break

                if s[i] == s[j]:
                    if j - i < 3:
                        dp[i][j] = True
                    else:
                        dp[i][j] = dp[i+1][j-1]

                if dp[i][j] and L > max_len:
                    max_len = L
                    start = i

        return s[start : start + max_len]

1143. 最长公共子序列 (LCS) - Medium

核心思路

比较两个字符串 text1 和 text2：

  1. 如果 text1[i-1] == text2[j-1]，当前字符属于公共子序列：dp[i][j]=dp[i−1][j−1]+1dp[i][j] = dp[i-1][j-1] + 1dp[i][j]=dp[i−1][j−1]+1。
  2. 如果不等，则尝试放弃其中一个字符：dp[i][j]=max⁡(dp[i−1][j],dp[i][j−1])dp[i][j] = \max(dp[i-1][j], dp[i][j-1])dp[i][j]=max(dp[i−1][j],dp[i][j−1])。

代码实现 

class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        # 多出一行一列处理边界（空字符情况）
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i-1] == text2[j-1]:
                    dp[i][j] = dp[i-1][j-1] + 1
                else:
                    dp[i][j] = max(dp[i-1][j], dp[i][j-1])

        return dp[m][n]

72. 编辑距离 - Medium/Hard

核心思路

这是二维 DP 的巅峰之作。将 word1 转为 word2，每一步有三种操作：

  • 插入：对应 dp[i][j−1]+1dp[i][j-1] + 1dp[i][j−1]+1
  • 删除：对应 dp[i−1][j]+1dp[i-1][j] + 1dp[i−1][j]+1
  • 替换：对应 dp[i−1][j−1]+1dp[i-1][j-1] + 1dp[i−1][j−1]+1

如果当前字符相等，则不需要操作，直接继承 dp[i−1][j−1]dp[i-1][j-1]dp[i−1][j−1]。

代码实现 

class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        # 边界初始化
        for i in range(m + 1): dp[i][0] = i
        for j in range(n + 1): dp[0][j] = j

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i-1] == word2[j-1]:
                    dp[i][j] = dp[i-1][j-1]
                else:
                    # 三种操作取最小值
                    dp[i][j] = 1 + min(dp[i-1][j],    # 删除
                                       dp[i][j-1],    # 插入
                                       dp[i-1][j-1])  # 替换
        return dp[m][n]
