# 动态规划 (Dynamic Programming)


## [70. 爬楼梯 - Easy](https://leetcode.cn/problems/climbing-stairs/)

### 核心思路
这其实是一个**斐波那契数列**变种。
1.  **递推公式**：$dp[i] = dp[i-1] + dp[i-2]$。
2.  **Python 优化**：我们可以利用 Python 的变量交换特性 `a, b = b, a + b`，直接在 $O(1)$ 空间内完成计算。

### 代码实现

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2: return n

        # a 代表 dp[i-2], b 代表 dp[i-1]
        a, b = 1, 2
        for _ in range(3, n + 1):
            a, b = b, a + b  # 状态同步更新
        return b
```

## [118. 杨辉三角 - Easy](https://leetcode.cn/problems/pascals-triangle/)

### 核心思路
每一行的两端是 `1`，中间元素 $row[j] = prev\_row[j-1] + prev\_row[j]$。

### 代码实现

```python
class Solution:
    def generate(self, numRows: int) -> list[list[int]]:
        res = []
        for i in range(numRows):
            # 初始化当前行全部为 1
            row = [1] * (i + 1)
            # 填补中间位置的数值
            for j in range(1, i):
                row[j] = res[i-1][j-1] + res[i-1][j]
            res.append(row)
        return res
```

## [198. 打家劫舍 - Medium](https://leetcode.cn/problems/house-robber/)

### 核心思路
对于每间房，只有“偷”或“不偷”两个选择：
* **状态方程**：$dp[i] = \max(dp[i-1], dp[i-2] + nums[i])$。
* **空间优化**：只需两个变量记录前两步的最优解。

### 代码实现

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        prev_max = 0  # dp[i-2]
        curr_max = 0  # dp[i-1]

        for x in nums:
            # 这里的交换逻辑是 Python 解决 DP 的杀手锏
            prev_max, curr_max = curr_max, max(curr_max, prev_max + x)

        return curr_max
```

## [279. 完全平方数 - Medium](https://leetcode.cn/problems/perfect-squares/)

### 核心思路
典型的**完全背包问题**：
* **状态方程**：$dp[i] = \min(dp[i], dp[i - j^2] + 1)$，其中 $j^2 \le i$。

### 代码实现

```python
class Solution:
    def numSquares(self, n: int) -> int:
        # 初始化为最大值（和为 i 最多需要 i 个 1）
        dp = [i for i in range(n + 1)]

        for i in range(2, n + 1):
            j = 1
            while j * j <= i:
                dp[i] = min(dp[i], dp[i - j * j] + 1)
                j += 1
        return dp[n]
```

## [322. 零钱兑换 - Medium](https://leetcode.cn/problems/coin-change/)

### 核心思路
* **状态方程**：$dp[i] = \min(dp[i], dp[i - coin] + 1)$。
* **初始化**：使用 `float('inf')` 表示无法达成，这是 Python 处理无穷大的地道方式。

### 代码实现

```python
class Solution:
    def coinChange(self, coins: list[int], amount: int) -> int:
        # 初始化 dp 数组，amount + 1 也是一种常用的占位符
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0

        for coin in coins:
            for i in range(coin, amount + 1):
                dp[i] = min(dp[i], dp[i - coin] + 1)

        return int(dp[amount]) if dp[amount] != float('inf') else -1
```

## [139. 单词拆分 - Medium](https://leetcode.cn/problems/word-break/)

### 核心思路
利用 `Set` 进行 $O(1)$ 的单词查找。$dp[i]$ 表示长度为 $i$ 的前缀是否可以拆分。

### 代码实现

```python
class Solution:
    def wordBreak(self, s: str, wordDict: list[str]) -> bool:
        word_set = set(wordDict)
        dp = [False] * (len(s) + 1)
        dp[0] = True

        for i in range(1, len(s) + 1):
            for j in range(i):
                # 如果前 j 个字符 OK 且剩下的 [j:i] 在字典里
                if dp[j] and s[j:i] in word_set:
                    dp[i] = True
                    break
        return dp[len(s)]
```

## [300. 最长递增子序列 - Medium](https://leetcode.cn/problems/longest-increasing-subsequence/)

### 核心思路
1.  **基础 DP** ($O(n^2)$): $dp[i] = \max(dp[j]) + 1$。
2.  **贪心 + 二分** ($O(n \log n)$): Python 的 `bisect` 模块可以极简地实现这个逻辑。

### 优化版代码实现

```python
import bisect

class Solution:
    def lengthOfLIS(self, nums: list[int]) -> int:
        tails = []
        for x in nums:
            # 找到第一个大于等于 x 的位置
            idx = bisect.bisect_left(tails, x)
            if idx == len(tails):
                tails.append(x)
            else:
                tails[idx] = x
        return len(tails)
```

## [152. 乘积最大子数组 - Medium](https://leetcode.cn/problems/maximum-product-subarray/)

### 核心思路
由于负负得正，必须同时维护**当前最大值**和**当前最小值**。

### 代码实现

```python
class Solution:
    def maxProduct(self, nums: list[int]) -> int:
        res = cur_max = cur_min = nums[0]

        for i in range(1, len(nums)):
            x = nums[i]
            # 当遇到负数时，最大最小值会互换
            if x < 0:
                cur_max, cur_min = cur_min, cur_max

            cur_max = max(x, cur_max * x)
            cur_min = min(x, cur_min * x)

            res = max(res, cur_max)
        return res
```

## [416. 分割等和子集 - Medium](https://leetcode.cn/problems/partition-equal-subset-sum/)

### 核心思路
转化成 **0-1 背包**：寻找和为 `sum / 2` 的子集。

### 代码实现

```python
class Solution:
    def canPartition(self, nums: list[int]) -> bool:
        total_sum = sum(nums)
        if total_sum % 2 != 0: return False

        target = total_sum // 2
        # dp[i] 表示和为 i 是否可能
        dp = [False] * (target + 1)
        dp[0] = True

        for num in nums:
            # 逆序遍历，防止重复计算
            for j in range(target, num - 1, -1):
                dp[j] = dp[j] or dp[j - num]
            if dp[target]: return True

        return dp[target]
```

## [32. 最长有效括号 - Hard](https://leetcode.cn/problems/longest-valid-parentheses/)

### 核心思路
使用**栈**存下标是最直观的 Python 实现方式。

### 代码实现

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        stack = [-1] # 哨兵
        max_len = 0

        for i, char in enumerate(s):
            if char == '(':
                stack.append(i)
            else:
                stack.pop()
                if not stack:
                    stack.append(i) # 更新新的哨兵起始位
                else:
                    max_len = max(max_len, i - stack[-1])
        return max_len
```
